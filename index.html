üìö About This Project

This is a demonstration of running a PyQt5 desktop application entirely in the browser using Docker, Guacamole, and Cloudflare Tunnels. No installation required!

Technologies Used:

  • PyQt5 - Modern GUI framework for Python
  • Docker - Containerized environment
  • Apache Guacamole - Clientless remote desktop gateway
  • Cloudflare Tunnel - Secure public access
  • Ubuntu Desktop - Linux desktop environment

Features:

  • Climate data analysis and visualization
  • Real-time data processing
  • Interactive plots and maps
  • Export results and reports
